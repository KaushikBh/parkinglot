# parkinglot Project

## Dev Machine Setup


### Getting the source code

Project is on [Github](https://github.com/KaushikBh/parkinglot). This is a public directory. To clone this directory on dev machine run git clone https://github.com/KaushikBh/parkinglot .




### Installing Java 7
 
 


### Setting up the project in Eclipse 

  * Open Eclipse
  * import .project from the directory you cloned into.

### To run the project 
From command line -
you can export parkinglot jar from eclipse & run java - jar parkinglot.jar
From eclipse -
Run Garage as java application.


Copy the output of that last command, add it your Github account.
